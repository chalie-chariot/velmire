## 재화 시스템 설계 문서

### 1. 핏방울 🩸
- 획득: 혈체 처치 시 3개 드롭
- 드롭 연출: 관 반대 방향으로 튀김 (BURST) → 관으로 흡수 (SUCK)
- 용도:
  - 노드 소환 비용
  - 노드 업그레이드 비용
- 특징: 전투 중 실시간으로 획득/소비되는 주요 전투 재화

### 2. 에센스 ✨
- 획득:
  - 자동화 시설 설치 후 시간당 자동 적립
  - 슈퍼챗 이벤트 (확률적 랜덤 발생, 우하단 알림창 연출)
  - 조력자 NPC가 확률적으로 지원
- 용도:
  - 노드 합체 비용
  - 고급 시너지 해금
- 특징: 전투보다 방송/자동화 컨셉에 연동된 패시브 재화
  - 슈퍼챗 발생 시 시청자 이름 + 금액 + 획득 에센스량 알림창 표시
  - 0일 때 UI 반투명, 1 이상 시 완전 표시

### 3. 특수재화 💎
- 획득:
  - 특정 조건 달성 시 지급
  - 희귀 드롭 (낮은 확률)
- 용도:
  - 희귀 시너지 해금
  - 특수 노드 합체
- 특징: 희귀도가 높아 신중하게 사용해야 하는 프리미엄 재화
  - 0일 때 UI 반투명, 1 이상 시 완전 표시
