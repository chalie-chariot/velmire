# Godot 4.x — 3D 카메라(또는 노드)가 정수 단위로만 움직일 때

## 1. 씬 트리에서 **잠금(Lock)** 풀기

- 왼쪽 **씬** 도크에서 `Camera3D` 선택
- 노드 이름 **오른쪽**에 **자물쇠 아이콘**이 켜져 있으면, 클릭해서 **잠금 해제**  
  → 잠긴 노드는 선택/이동이 막히거나 이상하게 동작할 수 있습니다.

## 2. **스냅(Snap)** 끄거나 간격 줄이기

정수(1, 2, 3…)만 움직이는 느낌이면 **이동 스냅 간격이 1**로 잡혀 있는 경우가 많습니다.

### 방법 A — 스냅 끄기 (가장 빠름)

- **3D 뷰포트** 상단 툴바에서 **자석(Snap)** 아이콘을 **끔**  
  (켜져 있으면 그리드에 맞춰 끊어서 움직입니다.)

### 방법 B — 스냅은 쓰되 **미세 간격**으로

- 메뉴 **Transform → Configure Snap…** (또는 뷰포트 상단 스냅 관련 메뉴)
- **Translate Snap** 값을 예:
  - `0.1` — 조금 촘촘함  
  - `0.01` 또는 `0.001` — **미세 조정**에 적합  

## 3. 숫자로 직접 넣기 (소수 허용)

- `Camera3D` 선택 후 **인스펙터** → **Transform**
- **Position** / **Rotation**에 **소수** 입력 (예: `3.531`, `3.552`)  
  → 여기서는 스냅과 무관하게 정확히 들어갑니다.

## 4. 이동 툴 사용 시 팁

- **W** (이동), **E** (회전), **R** (스케일)로 기즈모 조작
- 스냅이 켜져 있을 때 **Shift**를 누르면(버전에 따라) **더 작은 단위**로 움직이는 동작이 있을 수 있습니다.  
  → 안 되면 **2번**에서 스냅 간격을 `0.01` 이하로 낮추는 것이 확실합니다.

---

**요약:** 자물쇠 해제 → 자석 끄면 정수 느낌이 사라지는 경우가 많고, 필요하면 스냅 간격을 `0.01` 이하로 두면 미세 조정이 편합니다.
